당신의 AI가 두뇌, CareerMate는 커리어 서랍장

CareerMate 안에는 AI가 들어 있지 않습니다.
생각하는 일은 당신의 AI가, 데이터 보관은 CareerMate가 맡습니다.

01

당신의 AI와 대화

평소 쓰던 Claude에게 말로 시키면 됩니다. 분석·글쓰기 같은 “생각”은 전부 당신의 AI가 합니다.

02

안전하게 연결

표준 연결 방식(MCP)이 AI와 CareerMate를 잇는 다리가 됩니다. AI가 당신의 커리어 기록을 직접 읽고 정리해 저장합니다.

03

이 컴퓨터에 저장

모든 데이터는 로컬 SQLite(~/.careermate)에만 저장됩니다. 외부 서버로 전송하지 않고, 대시보드에서 눈으로 볼 수 있습니다.

지원 한 건을 끝까지 함께 처리

프로필 정리부터 면접 준비까지, AI와 대화만으로 이어집니다.

프로필·이력서 구조화업로드한 파일을 읽어 경력·프로젝트·스킬로 정리·보관.
채용공고 파싱·저장붙여넣은 공고 텍스트에서 회사·직무·자격요건을 추출.
적합도(핏) 분석내 경력과 공고를 비교해 강점·보완점·매칭 키워드 도출.
맞춤 자소서 + 버전 관리공고별 자기소개서를 버전으로 쌓고 비교, 파일로 내보내기.
지원 상태 8단계 관리작성 중 → 지원 → 서류 합격 → 면접 → 최종까지 칸반으로.
면접 준비서류 합격 시 예상 질문·STAR 답변·1분 자기소개 초안 생성.

추천 · 앱에서 한 문장 내 AI 앱에서 빈 폴더 잡고 한 문장

Claude 데스크톱의 “Code” 탭처럼 폴더를 잡고 작동하는 AI 앱이면 별도 터미널 없이 앱 안에서 끝납니다(Codex는 CLI 권장 — 데스크톱 앱은 등록이 사라질 수 있습니다). 작업 폴더 하나를 정하고 아래 문장만 주면, 에이전트가 공개 안내(careermate.life/llms-install.txt)를 읽고 그 폴더 안에 설치·연결합니다. 데이터까지 그 폴더에 모여(.cm/data) 백업·이동이 폴더 하나로 끝.

  1. 1

    앱 열고 빈 폴더 선택

    Claude 데스크톱 → Code 탭에서 빈 폴더를 작업 폴더로 선택(Pro 이상 구독, Windows는 Git 먼저). Codex는 빈 폴더에서 codex(CLI, ChatGPT Plus 이상)로 실행하세요 — 데스크톱 앱은 등록이 사라질 수 있어 권장하지 않습니다.

  2. 2

    이 문장 붙여넣기

    💬 위에서 연 AI 앱의 대화창에 붙여넣으세요 — 터미널 명령이 아니에요. 이 문장은 공개 안내 페이지만 읽어 설치하며, 당신의 이력서·개인정보는 외부로 올라가지 않아요.

    https://careermate.life/llms-install.txt 를 읽고, 그 안내대로 CareerMate를 이 폴더에 설치하고 연결해줘. 데이터도 이 폴더의 .cm/data 안에 저장하고, Node 설치처럼 시스템에 바꿀 게 있으면 실행 전에 먼저 알려줘. 설치가 끝나면 앱 재시작 방법과 이력서를 주는 법, 대시보드 주소(http://127.0.0.1:4319)도 알려줘.

    준비물은 Node.js 22.5 이상 하나뿐. 없으면 에이전트가 먼저 확인하고 설치를 도와줍니다. 데이터는 작업 폴더의 .cm/data에만 저장됩니다. (Codex CLI는 설치 단계에서 네트워크 허용을 한 번 물을 수 있어요 — 허용하면 됩니다.)

  3. 3

    재시작 후 확인

    AI 앱을 완전히 종료했다 다시 켜고(처음 한 번 승인을 물으면 허용), “커리어메이트 연결됐는지 확인해줘”라고 말해 보세요. Codex는 /mcp로도 확인됩니다.

터미널 · 한 줄 설치 npx 한 줄로 자동 연결

준비물은 Node.js 22.5 이상 하나. 감지된 AI 앱(Claude Desktop·Codex·Cursor)의 MCP 설정에 CareerMate를 자동 등록합니다.

  1. 1

    한 줄 실행

    터미널(Windows는 PowerShell)에 붙여넣고 실행하세요. 클론도 빌드도 필요 없습니다. node --version이 22.5 미만이면 nodejs.org에서 LTS를 먼저 설치하세요.

    npx -y careermate init

    특정 클라이언트만 등록하려면 npx -y careermate init --client cursor, 설정 JSON·명령만 보고 직접 붙여넣으려면 npx -y careermate init --print. (클라이언트: claude · claude-code · codex · cursor) 참고: Cursor는 MCP가 Pro 기능으로 표기돼 무료(Hobby)에선 제한될 수 있어요.

  2. 2

    AI 앱 재시작 후 확인

    연결한 AI 앱을 완전히 종료했다 다시 켜고, “커리어메이트 연결됐는지 확인해줘”라고 말해 결과가 돌아오면 정상입니다.

  3. 3

    대시보드 열기 (선택)

    내 데이터를 눈으로 보려면 npx -y careermate start 실행 → http://127.0.0.1:4319. 종료는 Ctrl+C.

Claude Desktop · .mcpb 설치 파일로 추가 (터미널 없이)

터미널·Node 없이 파일 하나로 끝나지만, Claude Desktop 내장 Node 버전에 따라 불안정할 수 있습니다. 잘 안 되면 위 에이전트가 직접 설치 또는 터미널 한 줄을 쓰세요.

  1. 1

    설치 파일 받기

    아래 버튼으로 careermate.mcpb를 내려받으세요. (약 2 MB)

    careermate.mcpb 다운로드
  2. 2

    Claude Desktop 설정에서 추가

    Claude Desktop → Settings(설정) → Extensions(확장)에서 Advanced settings(고급 설정)를 누르면 나오는 Install extension…으로 내려받은 careermate.mcpb를 선택하세요.

    더블클릭·드래그는 Windows에서 파일 연결이 안 잡혀 반응이 없을 수 있으니, 위 설정 메뉴 경로를 쓰세요. (확장을 처음 설치하면 Advanced settings를 먼저 눌러야 버튼이 보입니다.)

  3. 3

    재시작하고 확인

    Claude Desktop을 완전히 종료했다 다시 켜고, “커리어메이트 연결됐는지 확인해줘”라고 말해 보세요.

수동 등록 · 설정 파일 위치 Cursor·Cline·Windsurf

npx -y careermate init --print로 설정 JSON·명령만 출력해 클라이언트에 직접 추가할 수 있습니다. 클라이언트별 설정 파일 위치는 아래 탭을 참고하세요.

Windows: %APPDATA%\Claude\claude_desktop_config.json · macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
(Settings → Developer → Edit Config 로도 열 수 있습니다. npx -y careermate init이 이 파일을 자동으로 수정합니다.)

프로젝트별: .cursor/mcp.json · 전역: ~/.cursor/mcp.json — Cursor·Cline·Windsurf 등 대부분 같은 형식입니다. 추가 후 클라이언트를 다시 시작하세요.

문제가 생겼을 때 포트 충돌 · MCP 연결 실패 · 데이터 위치 · 초기화

포트가 이미 사용 중

4319가 막히면 자동으로 다음 빈 포트로 옮겨갑니다. 터미널에 출력된 실제 주소를 사용하거나, CAREERMATE_PORT 환경변수로 고정하세요.

AI에 MCP가 연결되지 않음

① npx -y careermate init이 정상 종료됐는지 ② node --version이 22.5 이상인지 ③ 클라이언트를 완전히 종료 후 재시작했는지 확인하세요. 폴더를 옮기거나 이름을 바꿨다면 등록된 경로가 깨질 수 있으니 init을 다시 실행하세요.

상태 점검

진단을 실행하면 Node 버전·데이터 폴더·DB 상태를 점검합니다.

npx -y careermate doctor

데이터 위치 / 초기화

기본 위치는 ~/.careermate/careermate.sqlite(Windows: %USERPROFILE%\.careermate). CAREERMATE_DATA_DIR로 변경 가능. 완전 초기화는 careermate.sqlite 삭제 후 재실행(백업 권장)입니다.

당신의 데이터는 당신 컴퓨터를 떠나지 않습니다

대시보드 서버는 127.0.0.1(이 컴퓨터)에만 바인딩되어 외부에서 접근할 수 없습니다.
MCP 서버가 외부로 내보내는 것은 (끌 수 있는) 최신 버전 확인뿐이며, 당신의 커리어 데이터는 전송하지 않습니다. 분석·글쓰기는 당신이 직접 고른 AI가 수행합니다.

  • 저장 위치 ~/.careermate/careermate.sqlite
  • 내보내기 자기소개서 등을 ~/.careermate/exports로
  • 삭제·관리 대시보드 설정(Settings)에서 직접

위 내용은 설치하는 앱 이야기예요. 지금 보고 계신 이 소개 웹사이트는 방문 통계를 보려고 쿠키 없는 익명 분석(Vercel Analytics)만 사용하며, 여기엔 어떤 커리어 데이터도 포함되지 않습니다.
