쓰는 AI와 컴퓨터만 고르면, 당신에게 맞는 설치 길 하나를 처음부터 알려드립니다.

1어떤 AI를 쓰고 있나요?

2컴퓨터는요?

위 두 가지를 고르면 아래에 당신에게 맞는 길이 나타납니다.

당신에겐 — Claude 데스크톱

Claude 데스크톱으로

두 가지 길이 있어요 — Code 탭(앱 안에서 한 문장·추천)과 파일 설치(터미널·Node 없이). 둘 다 같은 CareerMate에 연결됩니다.

방법 1 — Code 탭 (추천)

앱 안에서 끝나 별도 터미널이 필요 없습니다. Pro 이상 구독이 필요해요.

  1. 1

    Claude 데스크톱 설치 → 로그인

    claude.ai/download에서 받아 설치하고 본인 계정으로 로그인하세요.

  2. 2

    (Windows) Git 설치

    Code 탭을 처음 열면 앱이 "Git is required"라고 안내합니다.
    git-scm.com/download/win에서 설치하고 앱을 다시 켜면 됩니다. (Mac은 필요 없습니다.)

  3. 3

    Code 탭에서 빈 폴더 선택

    상단의 Code 탭을 누르고, 새 폴더 하나(예: 문서/CareerMate)를 작업 폴더로 고르세요.
    이 폴더가 CareerMate 작업실이 됩니다.

  4. 4

    이 문장 한 번 붙여넣기

    💬 Code 탭의 대화창에 붙여넣으세요 — 터미널 명령이 아니에요. 이 문장은 공개 안내 페이지만 읽어 설치하며, 당신의 이력서·개인정보는 외부로 올라가지 않아요.

    https://careermate.life/llms-install.txt 를 읽고, 그 안내대로 CareerMate를 이 폴더에 설치하고 연결해줘. 데이터도 이 폴더의 .cm/data 안에 저장하고, Node 설치처럼 시스템에 바꿀 게 있으면 실행 전에 먼저 알려줘. 설치가 끝나면 앱 재시작 방법과 이력서를 주는 법, 대시보드 주소(http://127.0.0.1:4319)도 알려줘.

    Node가 없어도 됩니다. 필요하면 Claude가 먼저 확인하고 설치까지 도와줍니다.
    직접 받으려면 아래 '필요한 프로그램 받기'의 Node.js를 보세요. 설치 중 뜨는 허용 창은 눌러 주세요.

  5. 5

    앱 재시작 → 확인

    Claude 앱을 완전히 껐다 켜고(처음 한 번 MCP 승인 허용), "커리어메이트 연결됐는지 확인해줘"라고 말하면 끝.

    도구가 안 보이면? 앱을 트레이까지 완전히 종료했다 다시 켜고, 처음 뜬 승인 창을 허용했는지 확인하세요. 그래도 안 되면 아래 방법 2(파일 설치)를 써 보세요.

📎 이력서 넣는 법: 가장 쉬운 방법은 이력서 내용을 대화창에 붙여넣기입니다. 또는 작업 폴더에 파일을 넣고 "이력서.pdf 읽어서 등록해줘"라고 하거나 채팅에 직접 첨부하세요. (스캔본·보안 PDF는 안 읽힐 수 있습니다 → 그땐 텍스트로 붙여넣으세요.)
방법 2 — 파일로 설치 (터미널·Node 없이)

Claude 데스크톱 채팅에 확장으로 붙이는 방식. 터미널·Node 없이 되고 무료·유료 어느 플랜이든 됩니다. 단 Claude 내장 Node 버전에 따라 불안정할 수 있어요 — 안 되면 방법 1을 쓰세요.

  1. 1

    설치 파일 받기

    .zip .mcpb

    .zip 압축을 풀어 폴더로 추가하세요. .mcpb 직접 추가는 일부 Claude Desktop 버전에서 실패할 수 있습니다.

  2. 2

    설정에서 추가

    Claude 데스크톱 → Settings → Extensions → Advanced settings → Install extension… 에서 압축을 푼 폴더를 추가하세요. (Windows에서 더블클릭·드래그는 반응이 없을 수 있으니 이 메뉴를 쓰세요.)

  3. 3

    재시작 → 확인

    앱을 껐다 켜고 "커리어메이트 연결됐는지 확인해줘". 도구가 보이면 끝.

📎 이력서 넣는 법: Claude 채팅창에 파일을 직접 첨부하면 됩니다(가장 쉬움).
당신에겐 — Codex CLI

ChatGPT 구독으로 Codex에서

ChatGPT는 Codex를 터미널(CLI)에서 씁니다 — 명령을 몇 줄 입력합니다.
Codex 데스크톱 앱은 등록이 사라질 수 있어 권장하지 않습니다.

  1. 1

    Node.js 설치

    Windows: PowerShell에 붙여넣기 →

    winget install OpenJS.NodeJS.LTS

    Mac: 터미널에 붙여넣기(Homebrew 없으면 nodejs.org에서 LTS 설치) →

    brew install node

    명령이 어려우면 nodejs.org에서 설치 파일을 받아 더블클릭(계속 '다음')해도 됩니다.
    설치 후 터미널을 새로 열고 node --version이 v22.5 이상인지 확인하세요. 터미널 여는 법은 아래 도움말을 참고하세요.

  2. 2

    Codex CLI 설치 → ChatGPT 로그인

    npm install -g @openai/codex

    설치 후 codex를 실행하면 ChatGPT 로그인 창이 열립니다(본인 구독 계정).

  3. 3

    빈 폴더에서 codex 실행 → 이 문장

    💬 실행한 codex 안에 붙여넣으세요 — AI에게 보내는 메시지예요(쉘 명령이 아님). 공개 안내만 읽어 설치하며, 당신의 데이터는 외부로 올라가지 않아요.

    https://careermate.life/llms-install.txt 를 읽고, 그 안내대로 CareerMate를 이 폴더에 설치하고 연결해줘. 데이터도 이 폴더의 .cm/data 안에 저장하고, Node 설치처럼 시스템에 바꿀 게 있으면 실행 전에 먼저 알려줘. 설치가 끝나면 앱 재시작 방법과 이력서를 주는 법, 대시보드 주소(http://127.0.0.1:4319)도 알려줘.

    설치 중 네트워크 허용을 물으면 한 번 허용하세요(npm 다운로드 때문).

  4. 4

    확인

    Codex 안에서 /mcp를 입력해 careermate가 보이면 연결 성공.

    안 보이면? node --version이 v22.5 이상인지, Codex를 껐다 켰는지 확인하세요. 그래도 안 되면 설치 전체의 문제가 생겼을 때를 참고하세요.

📎 이력서 넣는 법: 첨부 기능이 없으니, 작업 폴더에 파일을 넣고 "./이력서.pdf 읽어서 등록해줘"라고 하세요. PDF가 안 읽히면 내용을 텍스트로 붙여넣으세요.
당신에겐 — 기타 (Agent)

로컬 접근 가능한 AI 채팅에서

Claude·ChatGPT가 아니어도, 내 컴퓨터에서 명령을 실행할 수 있는 AI 에이전트면 됩니다. 빈 폴더를 잡고 아래 한 문장을 그 AI의 대화창에 붙여넣으세요.

⚠️ 무료·약한 모델은 한글 품질·사용량 한계가 있을 수 있어요(글자가 깨지거나 한도에 빨리 걸림). 제대로 쓰려면 Claude Pro·ChatGPT Plus를 권장합니다. CareerMate 자체는 계속 무료 — 한계는 고른 AI 쪽 이야기예요. Claude 데스크톱을 쓴다면 맨 위 Q1에서 ‘Claude’를 고르면 나오는 ‘방법 2(파일 설치)’가 더 쉬울 수 있어요.

  1. 1

    명령 실행 가능한 AI를 빈 폴더에서 열기

    내 컴퓨터에서 파일·명령을 다룰 수 있는 AI 에이전트(데스크톱 앱이든 CLI든)를 준비하고, 새 폴더 하나를 작업 폴더로 잡으세요.

  2. 2

    이 문장 붙여넣기

    💬 그 AI의 대화창에 붙여넣으세요 — 터미널 명령이 아니에요. 공개 안내만 읽어 설치하며, 당신의 데이터는 외부로 올라가지 않아요.

    https://careermate.life/llms-install.txt 를 읽고, 그 안내대로 CareerMate를 이 폴더에 설치하고 연결해줘. 데이터도 이 폴더의 .cm/data 안에 저장하고, Node 설치처럼 시스템에 바꿀 게 있으면 실행 전에 먼저 알려줘. 설치가 끝나면 앱 재시작 방법과 이력서를 주는 법, 대시보드 주소(http://127.0.0.1:4319)도 알려줘.
  3. 3

    확인

    그 AI를 완전히 껐다 켜고(처음 한 번 승인을 물으면 허용), "커리어메이트 연결됐는지 확인해줘" 또는 /mcp로 확인하세요.

    안 보이면? node --version이 v22.5 이상인지, 앱을 완전히 껐다 켰는지 확인하세요. 막히면 설치 전체의 문제가 생겼을 때를 보세요.

📎 이력서 넣는 법: 작업 폴더에 파일을 넣고 파일명·경로로 지시. 안 읽히면 텍스트로 붙여넣기.

도움이 더 필요하면

터미널(명령창)이 처음이세요? — 여는 법 Codex 등 CLI를 쓸 때

터미널은 명령을 한 줄씩 입력해 실행하는 창입니다. 이렇게 엽니다:

  • Windows — 시작 버튼 → PowerShell 검색 → Windows PowerShell 실행. 붙여넣기는 마우스 우클릭 또는 Ctrl+V, 실행은 Enter.
  • Mac — ⌘+Space(스포트라이트) → Terminal 입력 → 실행. 붙여넣기는 ⌘+V, 실행은 Return.

순서는 항상 같아요: 명령의 [복사] 버튼 → 터미널에 붙여넣기 → Enter.

필요한 프로그램 받기 (링크 모음) Node · 앱 · 설치 파일
  • Node.js (LTS) — nodejs.org · 설치 파일을 받아 더블클릭(계속 '다음'). 터미널·CLI 경로에 필요.
  • Claude 데스크톱 — claude.ai/download · Code 탭 / zip 폴더 설치에 사용.
  • Git for Windows — git-scm.com/download/win · Windows에서 Claude Code 탭을 쓸 때만(Mac은 불필요).
  • CareerMate 설치 파일 — careermate.zip / careermate.mcpb · Claude 데스크톱 "채팅" 앱용.

받은 뒤 무엇을 하는지는 위에서 고른 "내게 맞는 길"의 단계를 따르면 됩니다.
