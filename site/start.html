처음이세요?

터미널이 뭔지 몰라도 괜찮아요

쓰는 AI와 컴퓨터만 고르면, 당신에게 맞는 설치 길 하나를 처음부터 알려드립니다.

1어떤 AI를 쓰고 있나요?

2컴퓨터는요?

위 두 가지를 고르면 아래에 당신에게 맞는 길이 나타납니다.

당신에겐 — Claude

Claude로 연결하기

Claude Desktop 앱에서 끝내는 길과 터미널로 쓰는 길이 있어요. 대부분은 방법 1이면 됩니다.

방법 1 — Claude Desktop 앱 Code 탭 추천

Claude Desktop 앱 안에서 작업 폴더를 열고 진행합니다. 별도 터미널이 필요 없습니다.

  1. 1

    Claude Desktop 앱 설치 → 로그인

    claude.ai/download에서 받아 설치하고 본인 계정으로 로그인하세요.

  2. 2

    Code 탭에서 빈 폴더 선택

    상단의 Code 탭을 누르고, 새 폴더 하나(예: 문서/CareerMate)를 작업 폴더로 고르세요.
    이 폴더가 CareerMate 작업실이 됩니다.

    Windows에서 "Git is required"가 뜰 때만 Git for Windows를 설치하고 Claude Desktop 앱을 다시 켜세요.

  3. 3

    이 문장 한 번 붙여넣기

    💬 Code 탭의 대화창에 붙여넣으세요 — 터미널 명령이 아니에요. 이 문장은 공개 안내 페이지만 읽어 설치하며, 당신의 이력서·개인정보는 외부로 올라가지 않아요.

    https://careermate.life/llms-install.txt 를 읽고, 그 안내대로 CareerMate를 이 폴더에 설치하고 연결해줘. 데이터도 이 폴더의 .cm/data 안에 저장하고, Node 설치처럼 시스템에 바꿀 게 있으면 실행 전에 먼저 알려줘. 설치가 끝나면 앱 재시작 방법과 이력서를 주는 법, 대시보드 주소(http://127.0.0.1:4319)도 알려줘.

    Node가 없어도 됩니다. 필요하면 Claude가 먼저 확인하고 설치까지 도와줍니다. 설치 중 뜨는 허용 창은 눌러 주세요.

  4. 4

    앱 재시작 → 확인

    Claude Desktop 앱을 완전히 껐다 켜고(처음 한 번 MCP 승인 허용), "커리어메이트 연결됐는지 확인해줘"라고 말하면 끝.

    도구가 안 보이면? 앱을 트레이까지 완전히 종료했다 다시 켜고, 처음 뜬 승인 창을 허용했는지 확인하세요.

📎 이력서 넣는 법: Code 탭 채팅에 파일을 첨부하는 게 가장 쉽습니다. 첨부가 안 되면 작업 폴더에 파일을 넣고 "이력서.pdf 읽어서 등록해줘"라고 하거나, 내용을 텍스트로 붙여넣으세요.
방법 2 — Claude Code CLI 터미널에 익숙하다면

Claude를 터미널에서 직접 실행합니다. 터미널이 처음이면 방법 1을 쓰세요.

  1. 1

    Node.js 설치

    Windows: PowerShell에 붙여넣기 →

    winget install OpenJS.NodeJS.LTS

    Mac: 터미널에 붙여넣기(Homebrew 없으면 nodejs.org에서 LTS 설치) →

    brew install node

    명령이 어려우면 nodejs.org에서 설치 파일을 받아 더블클릭(계속 '다음')해도 됩니다. 터미널 여는 법은 아래 도움말을 참고하세요.

    Windows에서 Git for Windows는 CLI 필수 단계가 아닙니다. Bash 관련 오류가 날 때만 설치해도 됩니다.

  2. 2

    Claude Code 설치 → 로그인

    npm install -g @anthropic-ai/claude-code

    설치 후 claude를 실행하면 로그인 창이 열립니다(본인 Pro·Max 계정).

  3. 3

    빈 폴더에서 claude 실행 → 이 문장

    💬 실행한 claude 안에 붙여넣으세요 — AI에게 보내는 메시지예요(쉘 명령이 아님). 공개 안내만 읽어 설치하며, 당신의 데이터는 외부로 올라가지 않아요.

    https://careermate.life/llms-install.txt 를 읽고, 그 안내대로 CareerMate를 이 폴더에 설치하고 연결해줘. 데이터도 이 폴더의 .cm/data 안에 저장하고, Node 설치처럼 시스템에 바꿀 게 있으면 실행 전에 먼저 알려줘. 설치가 끝나면 앱 재시작 방법과 이력서를 주는 법, 대시보드 주소(http://127.0.0.1:4319)도 알려줘.

    설치 중 네트워크·허용을 물으면 한 번 허용하세요(npm 다운로드 때문).

  4. 4

    확인

    claude 안에서 /mcp를 입력해 careermate가 보이면 연결 성공.

    안 보이면? node --version이 v22.5 이상인지, claude를 껐다 켰는지 확인하세요.

📎 이력서 넣는 법: 파일 첨부가 가능하면 채팅에 바로 첨부하세요. 터미널에서 첨부가 어렵다면 작업 폴더에 파일을 넣고 "이력서.pdf 읽어서 등록해줘"라고 하면 됩니다.

    이 문장 붙여넣기

도움이 더 필요하면

터미널(명령창)이 처음이세요? — 여는 법 CLI를 쓸 때

터미널은 명령을 한 줄씩 입력해 실행하는 창입니다. 이렇게 엽니다:

  • Windows — 시작 버튼 → PowerShell 검색 → Windows PowerShell 실행. 붙여넣기는 마우스 우클릭 또는 Ctrl+V, 실행은 Enter.
  • Mac — ⌘+Space(스포트라이트) → Terminal 입력 → 실행. 붙여넣기는 ⌘+V, 실행은 Return.

순서는 항상 같아요: 명령의 [복사] 버튼 → 터미널에 붙여넣기 → Enter.

필요한 프로그램 받기 Node · Claude · Codex
  • Node.js (LTS) — nodejs.org · 설치 파일을 받아 더블클릭(계속 '다음'). 터미널·CLI 경로에 필요.
  • Claude Desktop 앱 — claude.ai/download · Code 탭에서 작업 폴더를 열 때 사용.
  • Codex 앱 — developers.openai.com/codex/app · 작업 폴더를 열고 프롬프트를 붙여넣을 때 사용.
  • Git for Windows — git-scm.com/download/win · Windows에서 Claude Desktop 앱 Code 탭이 "Git is required"라고 막을 때만.

받은 뒤 무엇을 하는지는 위에서 고른 "내게 맞는 길"의 단계를 따르면 됩니다.
